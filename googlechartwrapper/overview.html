Java Google Chart Wrapper Java Libary for Googles Chart API which supports developers to generate charts programmatically within the java environment. Most features of the google API are supported and it is easily expandable to match own requirements.

Notes

The download of the API (googlechartwrapper.jar) is available at sourceforge and only needs to be include in the classpath of the project. file. The generated strings and therefore chart urls may need a connection to the Internet because the charts are ultimately rendered by the Google Chart API server or the Eastwood project.

Usage

If you are using the API for the first time you may want to start with the examples in the unittest package available on sourceforge. Each chart has its own class and every feature is added or removed via similar named methods as addXYZ and removeXYZ, as getXYZ

The following charts are supported in version 0.2:

  • {@link googlechartwrapper.BarChart}
  • {@link googlechartwrapper.GeographicMap}
  • {@link googlechartwrapper.GoogleOMeter}
  • {@link googlechartwrapper.LineChart}
  • {@link googlechartwrapper.PieChart}
  • {@link googlechartwrapper.QRCode}
  • {@link googlechartwrapper.RadarChart}
  • {@link googlechartwrapper.ScatterPlot}
  • {@link googlechartwrapper.VennDiagram}

The data displayed in the charts is encoded in various forms to minimize the length of the url. Some charts will react to a change of the encoder and the displayed result may vary in some forms. For details reference the google API documentation

Developers who are interested in understanding the technic behind this API (feature system, reflection mechanism) and want to develop own charts based on it should see the following classes:

  • {@link googlechartwrapper.AbstractChart}
  • {@link googlechartwrapper.coder.AbstractEncoder}
  • {@link googlechartwrapper.util.GenericAppender}
  • {@link googlechartwrapper.util.UpperLimitGenericAppender}
